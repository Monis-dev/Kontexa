FAQ

Common questions

Where are my notes stored?
By default, all notes are saved to your browser's local storage — meaning they live entirely on your device and are available offline. No account is required. If you log in with Google and upgrade to Pro, notes are additionally backed up to our cloud servers for cross-device sync.
Is my data safe in the cloud?
We take security seriously. However, ContextNote uses free cloud hosting which, in rare cases, could experience data loss. We always recommend treating your local device as your primary backup. You can export notes at any time to Markdown for safekeeping.
How does the AI assistant work?
The AI assistant uses your own Google Gemini API key, which you enter in the settings. This means your note content is sent to Google's Gemini API when you ask a question — but it is never used to train models or stored by us. Your API key is saved locally in your browser only.
Can I add notes without using the extension popup?
Yes. When you open any source page or folder in the dashboard, there is an "Add Note" button in the top right. This lets you write notes with a title, content, and optional source URL directly from the dashboard — no need to be on the actual webpage.
What happens to my notes if I log out?
When logging out, you get two options: keep your notes on the device (they remain in local storage, fully accessible) or wipe them from the device. Your cloud backup is unaffected. You can log back in any time to re-sync from the cloud.
Does ContextNote work on all websites?
ContextNote works on virtually all standard web pages. Some browser-restricted pages (like the Chrome Web Store itself or browser settings pages) cannot run extensions — this is a Chrome security restriction, not a limitation of ContextNote.
